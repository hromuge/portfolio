My journey

2017-2019

At this time, I was a big fan of a video game called "Minecraft", which is completely written in Java.
When playing this game, you have an endless possibility of modifying the game with so-called "mods". I also tried my hand at making a few mods, as well as setting up my own server so I could play with my friends. This has helped me made my first steps into the world of coding.

2016

In this year, we started to have computer science classes at school. I was already familiar with the concepts of coding, so I already had somewhat of a headstart when we started to solve simple problems in a simple visual language called "Scratch".

2014

My journey began at the age of 10 when I first signed up for a video game creation course during summer vacation. In this course, we used a game engine called "Construct 2".
This is a HTML-5 based engine that executes JavaScript code in the background. To make it easier for beginners, it's a visual-oriented engine that allows you to create games without writing actual code.
